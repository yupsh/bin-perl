# yup-perl

```
NAME:
   perl - perl command wrapper for yupsh

USAGE:
   perl [OPTIONS] SCRIPT

   Run SCRIPT over standard input via perl -e.

VERSION:
   dev

GLOBAL OPTIONS:
   --loop, -n       assume loop around script
   --print, -p      assume loop like -n but print line also
   --autosplit, -a  autosplit mode with -n or -p
   --help, -h       show help
   --version        print version information and exit
```
